# My Portfolio

A personal portfolio website built from scratch with HTML and CSS.

## Project Structure

```
my-portfolio/
├── index.html      # Main HTML file
├── css/
│   └── style.css   # All styles
├── .prettierrc     # Code formatting rules
└── README.md       # This file
```

## Running Locally

### Option 1: Live Server (Recommended)

1. Open the project in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

The site will open in your browser and auto-refresh when you make changes.

### Option 2: Simple Python Server

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

### Option 3: Just Open the File

Double-click `index.html` to open it directly in your browser. You'll need to manually refresh after changes.

## Customization

1. Edit `index.html` to update your content
2. Modify `css/style.css` to change colors, fonts, and layout
3. Replace placeholder text with your actual information
